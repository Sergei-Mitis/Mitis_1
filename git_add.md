## git add

**git add *[файл]*** — добавляет файл в индекс.

или если все файлы в списке должны быть доступны всем, у кого есть доступ к репозиторию, одна команда добавит все в ваш текущий каталог и его подкаталоги:

```bash=
git add .
```
